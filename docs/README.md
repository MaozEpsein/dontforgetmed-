# Landing page

This folder serves as the GitHub Pages site for Don't Forget Med.

**To publish:**
1. Push to `main`.
2. In GitHub → Settings → Pages, set source to `main` branch, `/docs` folder.
3. The site will be available at `https://maozep.github.io/dontforgetmed/`.

Update `index.html` download button and `SettingsScreen.kt`'s `DOWNLOAD_URL` to match your final URL.
